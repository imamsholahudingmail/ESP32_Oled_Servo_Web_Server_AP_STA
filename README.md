# ESP32_Oled_Servo_Web_Server_AP_STA
Proyek **ESP32 Oled Servo WebServer AP STA** merupakan sistem IoT yang menggabungkan ESP32, servo motor, dan layar OLED. Perangkat dapat dikendalikan melalui web server dengan dua mode koneksi, yaitu Access Point (AP) dan Station (STA), sehingga fleksibel untuk monitoring dan kontrol jarak dekat maupun melalui jaringan WiFi.
